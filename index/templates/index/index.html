Projekt Crux

GitHub

Ein Warenwirtschaftsprogramm, das für eine kleine bis mittelgroße Produktionsfirma geeignet ist.
Es beinhaltet die folgenden Funktionen:

  • Artikel- und Kundendatenbank
  • Betriebswirtschaftliche Vorgänge (z.B. Bestellungen, Lieferscheine, Rechnungen, usw.)
  • Eine einfache Maschinenplanung, automatisch erstellt anhand vorhandener Bestellungen

  • Verwendetes Framework: Django
  • Verwendete Sprachen: Python, JavaScript, HTML (CSS), SQL

Tabellarische Liste aller Kunden (in SQLite DB, aber flexibel austauschbar)

Einfache Übersicht über die betriebswirtschaftlichen Vorgänge und beispielhaft eine daraus erzeugte Rechnung
